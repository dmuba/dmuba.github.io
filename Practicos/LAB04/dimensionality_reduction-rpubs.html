Reduccin de dimensionalidad: Low Variance Factor

Se eliminan los datos con escasa varianza, para ello se realiza el anlisis LVF:

# Tomamos los atributos numricos del dataframe
lvf<-na.omit(iris[,-c(5)])

# Primero normalizamos los datos (Min-Max) a un rango 0-1
for(i in 1:ncol(lvf)) {
  lvf[,i] <- (lvf[,i]-min(lvf[,i]))/(max(lvf[,i])-min(lvf[,i]))
}

# Calculamos la varianza para cada atributo y redondeamos a 4 decimales
varianzas<-round(apply(lvf, 2, var),4)

print(varianzas)

Sepal.Length  Sepal.Width Petal.Length  Petal.Width 
      0.0529       0.0330       0.0895       0.1009

Reduccin dimensionalidad: Atrib. Correlacionados

Primero debemos analizar si hay candidatos, podemos hacerlo grficamente con un heatmap:

library(gplots)
library(RColorBrewer)
# Reducing Highly Correlated Columns
dev.off()
ds.cor=cor(iris[,-c(5)], use="complete.obs")
heatmap.2(ds.cor,
          cellnote = round(ds.cor,1), 
          main = "Correlacin",
          notecol="black",     
          density.info="none", 
          trace="none",        
          margins =c(12,12),    
          col=brewer.pal('RdYlBu', n=5),  
          dendrogram="none",     
          Colv="NA")

Atributos Correlacionados (++)

Vamos a hacer el anlisis “a mano” y con la librera “caret”:

data.numeric<-na.omit(iris[,-c(5)])

# Calculo matriz de correlacion
matriz.correlacion<-cor(data.numeric)

# Verifico la Correlacin con la matrz
print(matriz.correlacion)

Atributos Correlacionados (+++)

Ahora lo hacemos con la librera Caret:

library(caret)

# Buscamos atributos con correlaciones superiores a 0.75
highlyCorrelated <- findCorrelation(matriz.correlacion, cutoff=0.75)

# Imprimimos los nombres de los atributos que cumplen con la condicin anterior
print(names(data.numeric[,highlyCorrelated]))

[1] "Petal.Length" "Petal.Width"

Luego deberamos analizar eliminar esos atributos.

Reduccin de dimensionalidad: Random Forests

Utilizamos la tcnica de Random Forest para ponderar los atributos en funcin de la importancia en la clasificacin. Primero corremos el modelo:

library(randomForest)

model_rf<-randomForest(Species ~ ., data=iris, na.action = na.omit)
