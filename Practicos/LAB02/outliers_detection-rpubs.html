Deteccin de Outliers mediante IRQ

  • Deteccin de outliers mediante el IRQ*1,5 (Rango intercuartil)
data.riq<-IQR(data)
print(data.riq)

[1] 59950

cuantiles<-quantile(data, c(0.25, 0.5, 0.75), type = 7)
print(cuantiles)

  25%   50%   75% 
18950 37550 78900

Deteccin de Outliers mediante IRQ (++)

Multiplicamos el cuantil 1 por 1.5 para determinar la barrera MENOR para la deteccin de outliers: 

outliers_min<-as.numeric(cuantiles[1])-1.5*data.riq
print(outliers_min)

[1] -70975

Y multiplicamos el cuantil 1 por 1.5 para determinar la barrera MAYOR para la deteccin de outliers: 

outliers_max<-as.numeric(cuantiles[3])+1.5*data.riq
print(outliers_max)

[1] 168825

Observemos el atributo ruidoso$Road_55dB "sin" outliers

plot(sort(data[data>outliers_min & data<outliers_max], decreasing = FALSE))


boxplot(sort(data[data>outliers_min & data<outliers_max], decreasing = FALSE))

Deteccin de Outliers mediante Desvos de la Media

Otra alternativa es realizar deteccin de outliers utilizando alguna medida de tendencia central.

Deteccin por N desvos de la media (En el ejemplo N=3):

N=3
data<-ruidoso$Road_55dB
desvio<-sd(data)
print(desvio)

[1] 484751.3

outliers_max<-mean(data)+N*desvio
print(outliers_max)

[1] 1613483

outliers_min<-mean(data)-N*desvio
print(outliers_min)

[1] -1295025

Observemos el atributo ruidoso$Road_55dB "sin" outliers

plot(sort(data[data>outliers_min & data<outliers_max], decreasing = FALSE))


boxplot(sort(data[data>outliers_min & data<outliers_max], decreasing = FALSE))

Deteccin de Outliers: Local Outlier Factor


Ahora abordamos, Local Outlier Factor [1] que es una tcnica multivariada, dado que tiene en cuenta varios atributos para la deteccin del outlier. La idea es calcular un score para cada instancia, el cual representa un “ndice” de densidad con respecto a los K vecinos mas cercanos.

# Cargamos la librera
library(Rlof)
data <- ruidoso
# Calculamos LOF para k=3 vecinos por instancia
data$score<-lof(data, k=3)


[1] https://cran.r-project.org/web/packages/Rlof/Rlof.pdf

Deteccin de Outliers: LOF (++)

Ahora analizamos el umbral a partir del cual consideramos que el score nos indica que es un outlier: 

# Ploteamos los scores
plot(sort(data$score, decreasing = FALSE))

Anlisis del umbral para determinar outlier

A partir del plot anterior, podriamos determinar que N=2 es un umbral de score “aceptable” para determinar si una observacin es un outlier o no: 

# Reemplazamos suponiendo un umbral de 1.8
umbral <-1.8
data.lof = data[data$score<umbral,]
plot(sort(data.lof$score, decreasing = FALSE))
